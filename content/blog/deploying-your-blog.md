# Deploying Your notro Blog

Once you've connected your Notion data source and verified the build locally, you're ready to deploy. notro generates a fully static site that can be hosted anywhere.

## Prerequisites

Before deploying, make sure:

1. `pnpm run build` succeeds locally.
2. Your environment variables (`NOTION_TOKEN`, `NOTION_DATASOURCE_ID`) are ready to configure on the hosting platform.

## Vercel

Vercel is the easiest option — it integrates directly with GitHub and automatically rebuilds on push.

1. Import your repository at [vercel.com/new](https://vercel.com/new).
2. Set the **Framework Preset** to **Astro**.
3. Add environment variables:
   - `NOTION_TOKEN`
   - `NOTION_DATASOURCE_ID`
4. Deploy.

To rebuild when Notion content changes, add a [Deploy Hook](https://vercel.com/docs/deployments/deploy-hooks) and trigger it from a Notion automation or a scheduled CI job.

## Netlify

1. Import your repository at [app.netlify.com](https://app.netlify.com).
2. Set **Build command** to `pnpm run build` and **Publish directory** to `dist`.
3. Add environment variables under **Site configuration → Environment variables**.
4. Deploy.

## Cloudflare Pages

1. Create a new Pages project and connect your repository.
2. Set **Build command** to `pnpm run build` and **Build output directory** to `dist`.
3. Add environment variables under **Settings → Environment variables**.
4. Deploy.

## Static export

All three platforms above serve the static `dist/` folder generated by `astro build`. You can also serve this folder from any static host (GitHub Pages, AWS S3 + CloudFront, etc.).

## Triggering rebuilds on content changes

Because notro fetches content at build time, you need to trigger a new build whenever you publish or edit a Notion page. Options:

- **Manual** — trigger a deploy from the hosting dashboard.
- **Webhook** — set up a Notion automation or cron job to POST to your platform's deploy hook URL.
- **Scheduled builds** — configure a scheduled CI job (e.g. GitHub Actions) to rebuild on a fixed interval.
